# ASUS-Prime-Z390M-PLUS Clover引导教程分享

>
>CPU：Intel Core i7-9700k
>
>显卡：Intel UHD Graphics 630
>
>内存：美国海盗船（USCORSAIR）复仇者LPX系列DDR4 3000 8GB✖️2

>系统盘：西部数据（WDC）WDS500G2B0A
>
>网卡：Intel AX200

>系统版本：macOS Catalina 10.15.4
>

CPU变频正常

核显驱动正常

主板HDMI输出正常
